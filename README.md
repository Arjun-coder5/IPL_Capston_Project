# IPL Data Analysis & Visualization

This project is based on analyzing an IPL (Indian Premier League) dataset using Python. The main focus is to clean, analyze, and visualize cricket data to understand team performance, player statistics, and match trends.

## Tools & Libraries Used

* NumPy – for numerical operations
* Pandas – for data cleaning and analysis
* Matplotlib – for basic visualizations
* Seaborn – for statistical and styled plots
* Plotly – for interactive charts
* Mffins – for advanced and creative visualizations

## Project Goals

* Analyze IPL match and player data
* Find insights like top teams, best players, win trends, and season-wise performance
* Create both static and interactive visualizations
* Make data easy to understand using graphs and charts

## Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Team and player performance analysis
* Match result and season trend visualization
* Interactive dashboards using Plotly

## Outcome

This project helps in understanding IPL data deeply using visualization and analysis. It is useful for beginners who want to learn data analysis and visualization with real-world sports data.
